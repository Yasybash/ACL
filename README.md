# Access Control System (File Utility)

Консольная программа для работы с файлами и управления их атрибутами безопасности в среде Windows (NTFS).

Реализует базовые операции контроля доступа: создание, чтение, запись файлов и изменение их атрибутов (read-only, hidden, normal).

## Возможности
- создание файла  
- чтение содержимого файла  
- запись данных в файл  
- изменение атрибутов файла в Windows:
  - только чтение (read-only)
  - скрытый файл (hidden)
  - обычный режим (read/write)

## Использование

При запуске программы доступно меню:
1 - create a file
2 - read file
3 - write to file
4 - changing attributes
0 - exit


### 1. Создание файла
- ввод имени файла
- создаётся пустой файл

### 2. Чтение файла
- ввод имени файла
- вывод содержимого в консоль

### 3. Запись в файл
- ввод имени файла
- ввод строки
- запись строки в файл

### 4. Изменение атрибутов
Поддерживаются режимы:
- **read-only** — запрет изменения файла  
- **hidden** — скрытие файла  
- **read/write** — снятие ограничений  

Изменение выполняется через WinAPI (`SetFileAttributes`).

## Принцип работы
Программа использует стандартные операции работы с файлами C и функции Windows API для изменения атрибутов безопасности файловой системы NTFS.

## Технологии
- C
- Windows API (`windows.h`)
- работа с файловой системой NTFS
